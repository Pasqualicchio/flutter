## 🔐 Funzionalità principali

- ✅ Login / registrazione utenti (SQLite)
- ✅ Upload immagini e file `.zip` (con estrazione automatica)
- ✅ Log automatico su file Excel (.xlsx) con anteprima miniatura
- ✅ UI responsive per visualizzare e scaricare i contenuti
- ✅ Filtri per tipo, ricerca istantanea, ordinamento e anteprime
- ✅ Modulo email integrato per invio file selezionati
- ✅ Supporto Windows/Linux/macOS (qualsiasi host con Python)
